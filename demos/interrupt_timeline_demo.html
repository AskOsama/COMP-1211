How Interrupts Work

1. Normal Execution: The CPU runs the user program (green).
2. I/O Request: Program requests data from I/O device.
3. I/O Transfer: Device performs the transfer independently (blue).
4. Interrupt Signal: When done, device sends interrupt (red flash).
5. ISR Execution: CPU pauses program, runs interrupt handler (orange).
6. Resume: After ISR completes, CPU resumes the original program.
