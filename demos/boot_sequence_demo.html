Understanding the Boot Process

When you press the power button, the CPU starts executing code from a fixed memory address (reset vector). This code is stored in ROM/firmware (BIOS or UEFI). The firmware performs hardware diagnostics (POST), locates a bootable device, loads the boot loader, which then loads the operating system kernel. Finally, the kernel initializes drivers and system services, preparing the computer for use.
